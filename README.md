# PHPTravels Automation Framework

A robust Selenium automation framework for testing the PHPTravels web application. Built with Java, TestNG, and Page Object Model (POM) design, it supports cross-browser testing, data-driven testing, and detailed reporting. Easily extendable for new test cases and integrates with Excel for test data management.

## Features
- Page Object Model structure
- Cross-browser support (Chrome, Firefox, Edge)
- Data-driven testing with Excel integration
- TestNG suite management and grouping
- Automated screenshots and logs
- HTML and XML test reports

## Usage
1. Clone the repository.
2. Configure browser and test data in `config.properties`.
3. Run tests using TestNG XML suites.
